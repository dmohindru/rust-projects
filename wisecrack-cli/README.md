# Rust Wisecrack cli app

A command line application to fetch daily dad jokes and motivational quotes of the day

## Options

### Fetch dad joke

```sh
-j, --joke
```

### Fetch Quote of the day

```sh
-q, --quote
```

### Format output

```sh
-o, --output text|json
```

### Config file option

```sh
-c, --config <file-path>
```
